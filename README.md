# ai-setup

CLI tool that installs and configures AI coding agents (Claude Code + OpenCode) with SDD orchestration, Engram memory, personas, MCP servers, skills, and permissions.

## Install

```bash
brew install ismarzdev/tap/ai-setup
```

## Usage

```bash
ai-setup install    # Interactive TUI installer
ai-setup sync       # Sync configuration to detected agents
ai-setup version    # Show version
```

## Supported agents

- Claude Code
- OpenCode
